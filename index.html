FAQ

คำถามที่พบบ่อย

ต้องมีความรู้เทคนิคในการติดตั้งไหม?
ไม่จำเป็น — แพ็กเกจ Full Package รวมบริการติดตั้งและ setup LINE OA ให้ทั้งหมด คุณแค่ต้องมี LINE Official Account ที่ verified แล้ว
ใช้งานกับ LINE OA แบบไหนได้บ้าง?
รองรับ LINE Official Account ทุกแพ็กเกจที่เปิดใช้ Messaging API ได้ ทั้งแบบ Free และ Premium
ข้อมูลลูกค้าและบิลเก็บไว้ที่ไหน?
ข้อมูลทั้งหมดเก็บในเซิร์ฟเวอร์ของคุณเองเท่านั้น ไม่มีการส่งข้อมูลออกไปยังบุคคลที่สาม มีระบบ Auto Backup เพื่อป้องกันข้อมูลสูญหาย
ถ้าบอทค้างหรือรีสตาร์ท ข้อมูลหายไหม?
ไม่หาย — ระบบ Auto Backup บันทึกทุกรอบและทุก state อัตโนมัติลงไฟล์ เมื่อรีสตาร์ทบอทจะโหลดข้อมูลล่าสุดขึ้นมาต่อได้ทันที
สลิปปลอมหรือสลิปซ้ำจัดการยังไง?
ระบบตรวจสลิปผ่าน EasySlip API โดยตรง ไม่ใช่แค่อ่านตัวเลข สลิปที่แคปหน้าจอหรือแก้ไขจะไม่ผ่าน และระบบจำ Transaction ID กัน ส่งสลิปซ้ำ
พร้อมใช้งานแล้ว?

ติดต่อสอบถามราคา รายละเอียด หรือสั่งซื้อได้เลย มีบริการติดตั้งและ customize ตามต้องการ
